## Quick Links

### Tools by Category

**Account Management**
- `get_balances` - Get account balances
- `get_active_orders` - List active orders

**Configuration**
- `get_currencies` - Get currency configuration
- `get_pairs` - Get trading pair configuration

**Public Market Data**
- `get_last_trades` - Get recent trades
- `get_order_book` - Get order book for a pair

**Trading**
- `place_order` - Place a new order
- `cancel_order` - Cancel an order
- `get_order` - Get order details

### Available Prompts

- `analyze-market` - Market analysis workflow
- `create-ladder-strategy` - Ladder trading strategy
- `portfolio-summary` - Portfolio valuation
- `risk-assessment` - Risk analysis
- `order-management` - Order management workflow
- `market-comparison` - Compare multiple pairs
- `price-alert-setup` - Price monitoring setup
